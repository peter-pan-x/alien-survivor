# 像素风格UI系统设计文档

## 🎮 设计理念

采用**8-bit/16-bit 复古像素游戏风格**，重现经典游戏的视觉体验。

### 设计参考

基于提供的像素艺术图标风格：
- 💀 **HP图标** - 绿色骷髅（生命值）
- 🐉 **ATK图标** - 橙色龙（攻击力）
- 👻 **MAG图标** - 浅蓝色幽灵（魔法/特殊能力）

## 🎨 视觉设计规范

### 颜色方案

```css
/* 主色调 */
背景: #1a202c (深灰蓝)
面板: #2d3748 (中灰蓝)
边框亮: #4a5568 (浅灰蓝)
边框暗: #1a202c (深色)

/* 功能色 */
生命值: #f56565 → #c53030 (红色渐变)
护盾: #4299e1 → #2c5282 (蓝色渐变)
经验: #fbbf24 → #d97706 (黄色渐变)
文字: #ffffff (白色)
标签: #cbd5e0 (浅灰)
```

### 字体规范

```css
主字体: 'Courier New', monospace
标题: 32-48px, bold, uppercase
正文: 14-18px, bold
标签: 12px, uppercase, letter-spacing: 1px
数字: 18-32px, bold, monospace
```

### 边框效果

**3D立体边框**:
```css
/* 凸起效果 */
border-color: #4a5568 #1a202c #1a202c #4a5568;
box-shadow: 
  inset -2px -2px 0 #1a202c,
  inset 2px 2px 0 #718096;

/* 按下效果 */
border-color: #1a202c #4a5568 #4a5568 #1a202c;
box-shadow: 
  inset 2px 2px 0 #1a202c,
  inset -2px -2px 0 #718096;
```

### 像素渲染

```css
image-rendering: pixelated;
image-rendering: -moz-crisp-edges;
image-rendering: crisp-edges;
-webkit-font-smoothing: none;
-moz-osx-font-smoothing: unset;
```

## 📐 组件设计细节

### 1. 像素按钮

**特点**:
- 3D边框效果（凸起/按下）
- 无圆角（完全方形）
- 像素字体
- 大写文字
- 无过渡动画（瞬间切换）

**状态**:
- 正常: 凸起边框
- 悬停: 边框颜色变亮
- 按下: 内嵌边框
- 禁用: 灰度 + 50%透明度

### 2. 像素进度条

**特点**:
- 固定高度 20px
- 渐变填充（顶部亮，底部暗）
- 黑色边框
- 数值显示在中央
- 平滑过渡（0.1s linear）

**类型**:
- `health`: 红色渐变
- `shield`: 蓝色渐变
- `exp`: 黄色渐变

### 3. 像素卡片

**特点**:
- 深色背景
- 3D边框
- 内嵌阴影
- 外阴影（深度感）
- 无圆角

### 4. 像素面板

**特点**:
- 更大的内边距
- 更强的阴影
- 适合作为容器

## 🎨 像素风格特性

### 1. 无抗锯齿

所有元素都使用像素渲染：
- 字体：无平滑
- 图片：像素化
- 边框：硬边缘

### 2. 高对比度

- 文字：白色 + 黑色描边
- 背景：深色
- 强调：亮色

### 3. 简单几何

- 无圆角
- 方形按钮
- 直线边框
- 网格背景

### 4. 复古配色

- 深色背景
- 高饱和度强调色
- 经典游戏配色

---

## 📊 对比：像素风格 vs 现代风格

| 特性 | 现代风格 | 像素风格 |
|------|---------|---------|
| **字体** | 平滑字体 | 像素字体 |
| **边框** | 圆角 | 方形 |
| **阴影** | 柔和模糊 | 硬边缘 |
| **动画** | 平滑过渡 | 瞬间切换 |
| **颜色** | 渐变 | 纯色/简单渐变 |
| **图标** | SVG矢量 | 像素图标 |
| **背景** | 模糊/渐变 | 网格/纯色 |

## 💡 设计原则

### 1. 保持像素感
- 所有元素都应该是"块状"的
- 避免平滑过渡
- 使用硬边缘

### 2. 高可读性
- 高对比度文字
- 清晰的标签
- 明确的数值显示

### 3. 复古氛围
- 经典配色
- 简单几何
- 像素字体

### 4. 功能优先
- 信息清晰
- 操作直观
- 响应迅速
