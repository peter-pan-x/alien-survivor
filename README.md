# 异星幸存者 (Alien Survivor)

一款顶部视角的无限流射击游戏，在异星球上对抗无尽的怪物！

## 🎮 游戏简介

你的星球已经毁灭，作为幸存者被派往异星寻找新的家园。在这个充满敌意的星球上，你需要击败不断涌现的怪物，通过升级提升自己的能力，尽可能存活更久！

## ✨ 游戏特色

- **无限流玩法**：敌人会不断涌现，难度逐渐提升
- **成长系统**：击败敌人获取经验，升级后选择技能强化自己
- **多样技能**：16种不同的技能供你选择，打造独特的战斗风格
- **Boss战斗**：每10级出现Boss，具备跳跃机制和紧张战斗音乐
- **精美视觉**：渐变色彩、粒子效果、光晕等视觉特效
- **触摸支持**：完美适配手机端，支持触摸操作
- **响应式设计**：自动适配手机和桌面浏览器

## 🎯 游戏玩法

- **控制方式**：移动鼠标/手指控制瞄准方向，虚拟摇杆移动
- **射击**：自动射击，无需手动操作
- **升级**：击败敌人获取经验，积累足够经验后升级
- **技能选择**：每次升级可从3个随机技能中选择1个
- **Boss战斗**：保持距离避免Boss跳跃，击败Boss获得丰厚奖励

## 🛠️ 技能系统

### 攻击类
- **攻击强化**：提升攻击力
- **速度强化**：提升攻击速度
- **射程强化**：增加攻击范围
- **多重射击**：增加子弹数量
- **子弹增幅**：增大子弹体积
- **暴击几率**：提升暴击概率
- **暴击伤害**：提升暴击伤害系数

### 防御类
- **生命强化**：提升最大生命值
- **护盾强化**：获得护盾保护

### 特殊类
- **穿透射击**：子弹可穿透敌人（史诗）
- **生命汲取**：击杀敌人恢复生命（史诗）
- **移动加速**：提升移动速度
- **轨道无人机**：获得环绕攻击无人机
- **闪电链**：定期释放连锁闪电
- **守护力场**：环状力场伤害并击退敌人
- **爆裂**：敌人死亡触发爆炸（稀有）

### 技能稀有度
- **史诗技能**：出现概率<5%（生命汲取、穿透射击）
- **稀有技能**：出现概率<10%（爆裂）

## 🎨 技术栈

- **React 19** - UI 框架
- **TypeScript** - 类型安全
- **Vite** - 构建工具
- **Tailwind CSS** - 样式框架
- **HTML5 Canvas** - 游戏渲染

## 📝 开发说明

游戏使用 Canvas 2D API 进行渲染，核心逻辑包括：

- 游戏循环 (Game Loop)
- 碰撞检测 (Collision Detection)
- 粒子系统 (Particle System)
- 状态管理 (State Management)
- 音频系统 (Audio System)
- Boss战斗系统 (Boss System)

## 🎵 音频系统

- **8-bit音效**：射击、击中、升级等音效
- **背景音乐**：普通探索时的轻松旋律
- **Boss音乐**：紧张战斗时的压迫感音乐
- **动态切换**：根据游戏状态自动切换音乐
